### Debug Tools

```bash
# Enable Terraform debug logging
export TF_LOG=DEBUG
export TF_LOG_PATH=./terraform-debug.log

# Enable IBM Cloud CLI debug
ibmcloud config --check-version false
ibmcloud config --trace true

# Enable Bob-Shell debug mode
bob config set debug true
bob config set log-level debug
```

## üìû Getting Help

### Quick Help

1. **Check the FAQ**: Most common questions are answered in the [FAQ section](https://bob-shell.dev/faq)
2. **Search Issues**: Check if your issue has been reported on [GitHub Issues](https://github.com/bob-ide/bob-shell/issues)
3. **Community Forums**: Ask questions in community forums
4. **Official Support**: Contact official support for critical issues

### Asking Good Questions

When asking for help, include:
- **Clear description** of the problem
- **Steps to reproduce** the issue
- **Expected vs actual** behavior
- **Environment details** (OS, versions, etc.)
- **Relevant code** or configuration
- **Error messages** (full text)
- **What you've tried** already
